Download Options

Get Marionette just the way you want it. We officially distribute via npm, bower, and static builds on this site.

Download Information v4.1.2

You can download Marionette either with npm or bower or as direct file download.

Install with bower

bower install marionette

Install with npm

npm install backbone.marionette

Static Builds

Pre-Packaged

These archives contain all of the files you need to get started with Marionette, including Backbone, jQuery and all other prerequisites.

These packages include the standard UMD version of both the non-minified and minified versions of Marionette.

*nix (.tar.gz)

backbone.marionette.tar.gz

Windows (.zip)

backbone.marionette.zip

Library (UMD)
